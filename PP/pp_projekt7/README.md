# Aplikacja do rejestrowania podróży

---

Aplikacja wykonana na potrzeby przedmiotu Paradygmaty programowania, UMG, sem. III, informatyka.

## Wymagania

Dane przechowywane w postaci listy krotek, gdzie pojedyncza krotka ma następującą postać:

```julia
(miasto, kraj, rok, ocenapodróży)
```

Aplikacja musi mieć następujące funkcję:

* filtrowanie po kraju
* mapowanie ocen
* wyciąganie lat podróży

Aplikacja musi umożliwiać śledzenie następujących statystyk:

* najczęściej odwiedzany kraj
* średnia ocena wyjazdów
